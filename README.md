# My First AI Chat Website

This is my first AI chat website project built with Flask and Python.

## Features

- Chat input box
- Send button
- AI response simulation
- Simple web interface

## Technologies Used

- Python
- Flask
- HTML

## How to Run

1. Install Flask

pip install flask

2. Run the app

python app.py

3. Open browser

http://127.0.0.1:5000
